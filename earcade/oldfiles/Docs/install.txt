To manually install....
 
Unzip and upload all the files/folders in the package keeping the folder structure the same as in the package..
e.g. Files in the \Sources folder go to <your forum>\Sources
Files not in folders go to <your forum> folder.

Create the new folders....
<your forum>\Games
<your forum>\cache
<your forum>\arcade
<your forum>\arcade\gamedata
<your forum>\tempGames
<your forum>\tempGames\gamedata

browse to <your forum>\installArcade.php to run the database installer.
