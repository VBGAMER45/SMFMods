[b]Global Announcements v1.0[/b]
https://www.smfhacks.com
by: vbgamer45

Display global announcements above topic listings on selected boards.

[b]Features:[/b]
[list]
[li]Create and manage global announcements from the admin panel[/li]
[li]Display announcements above topic lists on selected boards[/li]
[li]BBC editor support for announcement content[/li]
[li]Per-announcement board and membergroup targeting[/li]
[li]Comment system with BBC support[/li]
[li]View tracking and logging[/li]
[li]Force-read functionality to require members to view an announcement[/li]
[li]Convert announcements to/from forum topics[/li]
[li]Enable/disable toggle per announcement[/li]
[li]Custom display ordering[/li]
[li]Permission-based access (manage / comment)[/li]
[li]Who's Online integration[/li]
[/list]

[b]Compatibility:[/b] SMF 2.1.x

[b]Admin Settings:[/b]
Admin > Configuration > Global Announcements

[b]Permissions:[/b]
Admin > Members > Permissions
[list]
[li]Manage global announcements - Create, edit, delete, toggle announcements[/li]
[li]Comment on global announcements - Post and edit own comments[/li]
[/list]
