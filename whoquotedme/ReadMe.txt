By: vbgamer45

*******************************

Mod Information: 
For SMF 2.0.x, SMF 1.1.x

Shows the user a list of people that have quoted them.
Has built in option to rebuild quote history for all old posts


Install Information:
Install via the SMF's Package Manager via upload package.
