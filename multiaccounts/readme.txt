[b]Description:[/b]

Multi Accounts allows your forum members to create and manage multiple linked accounts from a single parent account. Members can switch between their accounts instantly without logging out and back in.

[b]Features:[/b]

[list]
[li]Create new linked accounts or link existing accounts[/li]
[li]Instant account switching via the account switcher widget[/li]
[li]Post as any linked account from the compose form[/li]
[li]Merge linked accounts (transfers all posts and data)[/li]
[li]Split linked accounts into independent accounts[/li]
[li]Reassign parent account status[/li]
[li]Share accounts between multiple parent accounts[/li]
[li]Search across all linked accounts[/li]
[li]Per-group account limits[/li]
[li]Full permission system (own/any for create, delete, merge, split)[/li]
[li]Child accounts cannot log in directly (must use parent account switcher)[/li]
[li]Admin panel to view and manage all linked accounts[/li]
[/list]

[b]Requirements:[/b]

[list]
[li]SMF 2.1.x[/li]
[/list]

[b]Installation:[/b]

[list=1]
[li]Upload the package via Admin > Package Manager > Download Packages[/li]
[li]Install the package[/li]
[li]Go to Admin > Members > Multi Accounts > Settings to enable the mod[/li]
[li]Configure permissions under Admin > Permissions[/li]
[/list]

[b]Usage:[/b]

[list=1]
[li]Members can manage their linked accounts via Profile > Multi Accounts[/li]
[li]Use the account switcher in the menu to switch between accounts[/li]
[li]When composing a post, use the "Post as" dropdown to post as a linked account[/li]
[/list]
