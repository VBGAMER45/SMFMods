=================

A comprehensive clan/ladder/league/tournament system for SMF 2.1.

Features:
- Clan management (create, join, manage members, logos)
- Ladder system with multiple ranking modes (ELO, Swap, RTH)
- Match challenges and result reporting with confirmation workflow
- Tournament brackets (single/double elimination, round-robin)
- League/season management with historical data
- Advanced player statistics (kills, deaths, assists, goals)
- MVP awards and leaderboards
- SMF 2.1 alert integration (10 alert types)
- Full admin panel for all system management
- Match finder / matchmaking queue
- Per-match and per-ladder player statistics
- Activity status tracking with visual indicators

Requirements:
- SMF 2.1.0 or higher
- PHP 7.0 or higher
- MySQL 5.6 or higher

Installation:
1. Upload the package through Admin > Packages
2. Install using the package manager
3. Configure via Admin > Configuration > Rivals
